# Volume Studio

אתר לאולפן הקלטות "ווליום" – שירים, קריינות, ג'ינגלים ועוד.

## טכנולוגיות
- Frontend: React
- Backend: Node.js + Express
- Database: MongoDB Atlas
- Deployment: Vercel/Netlify

## תכונות
- גלריה של תמונות, סרטונים ושירים
- הצגת חבילות ומחירים
- טופס יצירת קשר
- ניהול מדיה למנהל
- עיצוב חדשני בסגנון ורוד-זהב

## איך מריצים מקומית
```bash
npm install
npm start
